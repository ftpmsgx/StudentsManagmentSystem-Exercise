### 文件列表

***

| 文件名        | 功能                               |

| main.php      | 登录后的主页面                     |
| add.php       | 添加记录的页面文件                 |
| newRecord.php | 添加记录的处理文件，与上一文件关联 |

| update.php    | 编辑记录的处理文件，与上一文件关联 |
| delete.php    | 删除记录文件（无页面）             |
| search.php    | 搜索记录文件                       |
| t1.php        | 临时的文件                         |
